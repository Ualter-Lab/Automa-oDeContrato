# Gerador de Contrato - Walter Neto

Site estático para gerar contrato/recibo de prestação de serviços web em PDF.

## Novidades desta versão

- Campo para colar o resumo da calculadora de orçamento.
- Leitura automática de tipo de projeto, páginas, prazo, itens, valor cheio, desconto e total final.
- Forma de pagamento com seleção: Pix, cartão de crédito ou cartão de débito.
- Parcelamento padrão 40/30/30, mas editável.
- Observações automáticas para: Treinamento do cliente, Manutenção mensal e Atualizações futuras.
- Prévia do contrato, cópia do texto e geração de PDF.

## Como usar

Abra o arquivo `index.html` no navegador.

Para gerar PDF, mantenha conexão com internet, pois o site usa a biblioteca jsPDF via CDN.
